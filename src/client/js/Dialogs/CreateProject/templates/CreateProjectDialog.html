Creates a new project from an existing snapshot. It can either be a template file-seed available on the server or a branch from one of the existing projects. The new project will have a single commit and a branch (master) pointing to the commit.
Duplicate project
This will make a full copy of an existing project with its entire history. This includes all commits, branches and data-objects.
